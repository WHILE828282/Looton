# Game images

Place your real game photos in this folder manually (do not commit binary patches if your workflow blocks them).

Use the same structure for every game:
- `public/games/mlbb/cover.jpg` or `public/games/mlbb/cover.jpeg`
- `public/games/gta5/cover.jpg` or `public/games/gta5/cover.jpeg`
- `public/games/cs2/cover.jpg` or `public/games/cs2/cover.jpeg`
- `public/games/pubg/cover.jpg` or `public/games/pubg/cover.jpeg`

The UI automatically tries `.jpeg` and `.jpg` variants and then falls back to `/icon.svg`.
